A mobile-first, single-page conference website for DoctorCon 2026 with Kolkata/Bengali-inspired aesthetics.

## Structure
- `index.html`: Single-page site using Tailwind CSS (CDN) + vanilla JS for menu and committee list.

## How to view
Open `index.html` directly in a browser (no build step). If you want a local server for live reload, use any static server.

## Customize
- Update colors and typography in the Tailwind config in `index.html`.
- Swap hero video URLs and map embed in the respective sections.
- Replace committee list in the inline JS array.

## Notes
All forms are static placeholders. Wire them to a backend or form provider when needed.
